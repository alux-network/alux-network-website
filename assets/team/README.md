# ALUX 团队乐高头像 — 设计规范与新成员制作指南

## 文件说明（双轨制）

| 位置 | 用途 | 特点 |
|---|---|---|
| `script.js` 中的 `renderTeamPortrait()` | **网站运行时**，Team 页实际渲染 | 颜色走 CSS 变量，调色只改 `styles.css` |
| `assets/team/*.svg`（本目录） | **定稿导出版**，供网站以外复用（社交头像、文档、README 等） | 颜色写死，改色需重新导出 |

改设计时：先改内联版（script.js + styles.css）→ 确认效果 → 再导出覆盖本目录的静态 SVG。

## 新增成员步骤

以新成员 `alice` 为例：

1. **`script.js` → `renderTeamPortrait()`**，在四个字典各加一个 `alice` 键：
   - `lidPrints`：笔记本盖上的签名图案（她的技术标志，如某语言符号、图形）
   - `torsoDecor`：衣领/领带/拉链等上衣细节（画在笔记本图层之下，y96–104 之间可见）
   - `headAccessories`：发型、眼镜、胡子、帽子等（按本人照片做，最后绘制、覆盖在头之上）
   - `smiles`：嘴型（大笑/微笑/平静线；若胡子里含嘴则留空 `""`）
2. **`styles.css`**，复制一个 `.team-portrait-wrap--frank` 块改名 `--alice`，按配色规则填变量。
3. **`script.js` → `pageData.team`** 加入英文成员条目，并设置 `portrait: "alice"`。英文是唯一内容源；随后运行 `npm run i18n:scan`，在 `i18n/locales/*.json` 中完成四种语言的母语化校对。
4. 运行 `npm run i18n:apply` 与 `npm run i18n:verify`，再统一更新 HTML 中的资源缓存版本号。

## 几何规范（viewBox 220×220，人物居中 x=110）

- 卡片底：`rect x=10 y=10 200×200 rx=42` 奶油色 `#fffaf3`
- 光晕：左上 `(68,62) r50`，右下 `(154,154) r48`
- 头：`x87 y48 46×40 rx10`；眼 `(101,65)/(119,65) r2.7`；嘴基线 y75–78
- 颈：`x101 y88 18×8`；躯干梯形 `M83 96 L137 96 L143 142 L77 142`
- 笔记本：底座 `x85 y128 50×6`，屏背梯形 y102–128，转轴 y126.5；印花区约 x95–125 / y105–124
- 手：实心圆 `(87,130)/(133,130) r7.5`，肤色
- 胯 `x85 y142 50×11`；腿 `x87/x111 y153 22×37`；底板积木 y190 统一 `#a67951`

## 配色规则

- **肤色**（不用乐高黄）：亚洲标准 `#f1c9a4`，偏深 `#e4b389`（Ben 档），白皙粉调 `#efc9b4`（Tomislav 档）。同族只动半档，别拉大。
- **光晕/背景积木**：左上统一暖沙 `#a67951`（`--portrait-a`），右下为个人色（`--portrait-b`），个人色从站点莫兰迪色池选：绿 `#6ca07d`、藕紫 `#a88eac`、蓝灰 `#8d91b8`、金 `#bea16a`。
- **笔记本与上衣必须深浅对比**：深色衣服 → 浅色电脑（如 Frank 藏青衣+浅蓝灰电脑）；浅色衣服 → 深色电脑（如 Tom 米白衫+石墨电脑）。印花色 `--lego-print` 与盖子反差。
- 全部取色保持低饱和莫兰迪调，与官网奶油色系一致；避免纯白、纯黑、高饱和色。

## CSS 变量清单（每人一套）

```css
body[data-page="team"] .team-portrait-wrap--alice {
  --portrait-a: #a67951;   /* 左上光晕+积木，统一暖沙，勿改 */
  --portrait-b: #6ca07d;   /* 右下光晕+积木，个人色 */
  --portrait-c: #26364a;   /* 姓名缩写徽章底色 */
  --lego-skin: #f1c9a4;    /* 肤色 */
  --lego-laptop: #aebccf;  /* 笔记本盖 */
  --lego-laptop-dark: #7f93ad; /* 底座+转轴 */
  --lego-print: #26364a;   /* 盖上印花 */
  --lego-torso: #26364a;   /* 上衣（含手臂） */
  --lego-legs: #34455c;    /* 腿 */
  --lego-hip: #26364a;     /* 胯 */
}
```

## 导出静态 SVG

把内联版输出的 SVG 中所有 `var(--xxx)` 替换为该成员的实际色值，外层补上卡片底和两个光晕，存为 `assets/team/<name>.svg`。现有四个文件即为此格式，可直接参照。
